LRO Solar Radiation Pressure
=============================

This example demonstrates how to compute solar radiation pressure (SRP) accelerations
using the object-oriented interface of the pyRTX library.

This approach computes SRP accelerations on-the-fly using the spacecraft geometry
and solar position at each time step.
